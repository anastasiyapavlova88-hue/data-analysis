Bellabeat Smart Device Usage Analysis
Project Overview

This case study analyzes smart device usage data from Fitbit users to generate marketing insights for Bellabeat, a wellness company focused on women's health.

The goal of the project is to identify behavioral trends and provide data-driven marketing recommendations.

Business Task

Analyze smart device usage data to discover trends in user behavior and provide high-level marketing recommendations for Bellabeat.

Tools Used
SQL (BigQuery)
Tableau (Data Visualization)
Google Sheets (Data storage)
Dataset

Fitbit Fitness Tracker Data (Kaggle)
Time period: March 12 – May 11, 2016
Includes:

Daily activity
Steps
Sleep
Calories
Weight

Dataset contains data from 30 Fitbit users.

Key Insights
Users track activity regularly but rarely track sleep and weight
Average daily steps: 7,377 (below recommended 10k)
Average sleep: 2.09 hours recorded
Average sedentary time: 16.7 hours/day
Strong correlation between steps and calories (0.58)
Weak correlation between steps and sleep (0.11)

Main finding:
The biggest opportunity for Bellabeat is improving habit formation and long-term engagement.

Business Recommendations
Habit-building onboarding (21-day challenges)
Strong focus on sleep tracking features
Gamification (badges, streaks, challenges)
Segment-based marketing
